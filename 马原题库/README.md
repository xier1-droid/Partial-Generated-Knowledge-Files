# 马原题库

这是一组用于《马克思主义基本原理》题库提取、整理和刷题的资料集合。仓库中保留了原始的离线刷题页面、提取后的 Excel 汇总、题库提取说明，以及配套的提取指南压缩包。

## 目录内容

```text
马原题库/
├── README.md
├── 离线题库刷题.html
├── 题库汇总.xlsx
├── 题库提取使用说明.md
└── 学习通题库提取指南.zip
```

## 文件说明

### `离线题库刷题.html`

- 题库离线刷题页面
- 适合直接在浏览器中打开复习
- 本地使用，不依赖网络

### `题库汇总.xlsx`

- 题库汇总表
- 实际包含 3 张工作表：`复习题库`、`错题本`、`汇总`
- 适合做系统复习、查看错题和统计结果

### `题库提取使用说明.md`

- 题库整理和导出说明
- 讲清楚如何把本地 HTML 题库批量整理为 Excel
- 适合第一次接触这套工具时阅读

### `学习通题库提取指南.zip`

- 提取指南压缩包原件
- 作为原始资料保留，便于回看和分发

## 使用建议

1. 先看 `题库提取使用说明.md`，了解导出流程。
2. 再打开 `离线题库刷题.html` 进行练习。
3. 用 `题库汇总.xlsx` 做集中复习和错题回顾。
4. 如需保留原始说明包，可直接打开 `学习通题库提取指南.zip`。

## 备注

- 文件名保持原样，方便和本地资料一一对应。
- 如果后续还有新的题库导出内容，可以继续放在这个目录下，并沿用同样的命名习惯。
