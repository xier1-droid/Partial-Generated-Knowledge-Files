# 题库提取使用说明

这份说明面向零基础同学，目标是把你已经**合法导出的本地 HTML 题库文件**批量整理成一个方便复习的 Excel 文件。

本工具不会帮你登录平台、绕过限制或抓取你没有权限访问的内容。它只处理你已经保存在电脑上的 HTML 文件。

## 你会得到什么

运行脚本后，会生成一个 `xlsx` 文件，里面通常有 3 张表：

- `复习题库`：完整题目列表，方便逐题复习
- `错题本`：只保留错误和未答题目
- `汇总`：统计总题数、正确率、按题型统计

## 一、准备工作

### 1. 安装 Python

如果电脑还没有 Python，请先安装 Python 3.10 或更高版本。

安装后，在 PowerShell 里输入：

```powershell
python --version
```

如果能看到版本号，说明安装成功。

### 2. 安装依赖

脚本只需要一个额外依赖：`beautifulsoup4`

打开 PowerShell，输入：

```powershell
python -m pip install beautifulsoup4
```

如果提示安装成功，就可以继续。

### 3. 确认脚本文件在当前目录

本项目里已经有提取脚本：

- [extract_tiku.py](./extract_tiku.py)

你后面运行命令时，建议先切换到这个脚本所在的文件夹。

## 二、整理 HTML 文件

### 1. 文件名不需要统一

每个 HTML 文件名字不一样也没关系，例如：

- `第一章.html`
- `第二章单选.html`
- `期末模拟.html`

脚本不会依赖文件名，只会扫描目录里的 `.html` / `.htm` 文件。

### 2. 保持 HTML 和 `_files` 文件夹在一起

如果你是从浏览器“另存为网页”得到的文件，通常会同时生成一个同名的 `_files` 文件夹。

例如：

- `第一章.html`
- `第一章_files\`

这两者要放在同一个父目录下，不要拆开。

### 3. 推荐的目录结构

建议你把所有题库页面统一放到一个总目录里，例如：

```text
题库导出\
  第一章.html
  第一章_files\
  第二章单选.html
  第二章单选_files\
  第三章.html
  第三章_files\
  期末模拟.html
  期末模拟_files\
```

只要 HTML 文件都在这个总目录下，脚本就能批量找到它们。

## 三、批量提取题库

### 方法一：提取整个目录

如果你已经把所有 HTML 都放进一个目录，直接运行：

```powershell
python extract_tiku.py --input "D:\Download\OneDrive\桌面\题库导出" --output "D:\Download\OneDrive\桌面\学习\大二\马原题库\题库汇总.xlsx" --recursive
```

这个命令会：

- 递归扫描目录里的所有 HTML 文件
- 自动解析题目
- 导出成一个 Excel 文件

### 方法二：只提取单个 HTML 文件

如果你只想先试一个文件，可以这样运行：

```powershell
python extract_tiku.py --input "D:\Download\OneDrive\桌面\题库导出\第一章.html" --output "D:\Download\OneDrive\桌面\学习\大二\马原题库\第一章题库.xlsx"
```

### 方法三：输出到当前目录

如果你已经在脚本所在目录打开 PowerShell，也可以直接写文件名：

```powershell
python extract_tiku.py --input "D:\Download\OneDrive\桌面\题库导出" --output "题库汇总.xlsx" --recursive
```

## 四、如何运行最稳

建议按下面顺序做：

1. 把所有 HTML 放进一个总目录
2. 打开 PowerShell
3. 切换到脚本所在目录
4. 执行提取命令
5. 打开导出的 Excel 检查结果

例如：

```powershell
cd "D:\Download\OneDrive\桌面\学习\大二\马原题库"
python extract_tiku.py --input "D:\Download\OneDrive\桌面\题库导出" --output "D:\Download\OneDrive\桌面\学习\大二\马原题库\题库汇总.xlsx" --recursive
```

## 五、结果怎么看

导出的 Excel 里，一般会有这些列：

- `序号`
- `来源文件`
- `题号`
- `题型`
- `题目`
- `A` 到 `F`
- `我的答案`
- `正确答案`
- `结果`
- `解析`

### 你可以重点看这些内容

- `结果`：判断这道题是正确、错误还是未答
- `正确答案`：标准答案
- `解析`：如果题目里有解析，这里会显示
- `错题本`：只看需要重点复习的题目

## 六、常见问题

### 1. 提示 `No HTML files found`

说明脚本没找到 HTML 文件，常见原因有：

- 目录选错了
- 文件不是 `.html` / `.htm` 后缀
- HTML 文件不在你指定的目录里

解决办法：

- 检查输入目录
- 确认文件后缀
- 尽量把所有 HTML 放到一个总目录里再跑

### 2. 提示 `Missing dependency: beautifulsoup4`

说明依赖没装好，重新执行：

```powershell
python -m pip install beautifulsoup4
```

### 3. 导出的 Excel 打不开

一般是下面几个原因：

- 输出文件被别的软件占用
- 输出路径写错了
- 生成过程中中断了

解决办法：

- 关闭正在打开的 Excel
- 删除失败的输出文件后重新导出
- 换一个更简单的文件名，例如 `题库汇总.xlsx`

### 4. HTML 文件名字都不一样怎么办

不用管，名字不一样完全没关系。

脚本只看目录里的 HTML 文件，不要求统一命名。

### 5. `_files` 文件夹要不要删

不要删，保留原样就行。

它们是网页资源文件夹，虽然脚本主要读取 HTML，但最好让 HTML 和对应资源文件夹保持在同一目录中。

## 七、推荐的新手操作流程

如果你是第一次用，照这个顺序最省事：

1. 安装 Python
2. 安装 `beautifulsoup4`
3. 把所有 HTML 放进一个总目录
4. 打开 PowerShell
5. 运行批量导出命令
6. 打开导出的 `xlsx`
7. 先看 `汇总` 页，再看 `错题本`
